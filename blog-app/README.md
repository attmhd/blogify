## Installation

Clone project

```bash
  https://github.com/attmhd/blogify.git
```
Go to project directory

```bash
  cd simple-blog
```
Rename **env.example** to **.env**

Install depedencies

```bash
composer install
npm install
```
Generate Key

```bash
php artisan key:generate
```    

Migrate Database
```bash
php artisan migrate
```

Running project

```bash
php artisan serve
npm run dev
```
